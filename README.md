# Crypto Crash Game Frontend

This repository contains the frontend code for a **Crypto Crash Game** built with modern web technologies. The game allows users to experience a fun and interactive game based on the mechanics of cryptocurrency price movements, where players can "bet" on the crash point of a cryptocurrency value and try to predict the moment the price crashes.

## Features

- **Interactive UI**: Easy-to-use and responsive interface for users to interact with the game.
- **Real-time Data**: The game fetches real-time crypto price data to simulate price fluctuations.
- **Crash Prediction**: Players predict when the price will "crash" and can cash out their bet at any point.
- **Betting System**: Players can place bets based on the cryptocurrency price movements.
- **Leaderboards**: Display top winners, game history, and overall statistics.
- **WebSocket Support**: Live updates for the game, ensuring smooth gameplay experience.

## Tech Stack

- **Frontend**: 
  - ReactJS
  - Redux (for state management)
  - WebSockets (for real-time data streaming)
  - Tailwind CSS (for styling)
